# ComboBoxRemove

ComboBoxRemove is a component that lets you select/remove an option from a set list of options.

**behaviour**

- The options show the display value of the selected option.
- The options show when the ComboBox gains focus.
- The options stop showing when the ComboBox looses focus.
- When an option is selected, it must call onChange passing the option's value as the only parameter to the function and show the X icon.
- When X is clicked, then onChange is called with a null option.
