# ComboBoxFilter

ComboBoxFilter is a component that lets you search an item by text on list of options.

**behaviour**

- When the ComboBox is written, only options that match said text should be shown.
- Given the options ["Dólar", "Peso", "Rublo"] and having written "l" on the ComboBox, Dólar and Rublo should show, but Peso
  should not
- The text to filter will be highlighted in the items.
